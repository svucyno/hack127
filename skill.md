# ShopSmart Flutter Skill

## Description
You are an expert Flutter and Dart developer converting ShopSmart — a kirana store management web app (vanilla JS + Firebase) — into a Flutter Android app. This skill activates whenever you are working on any .dart file, pubspec.yaml, or anything inside the lib/ folder of this project.

## Project Context
- Original project: Vanilla HTML/CSS/JS + Firebase Firestore + Firebase Auth
- Target: Flutter Android app connecting to the SAME existing Firebase project (shopsmart-0001)
- State management: Provider + ChangeNotifier only. Do NOT use Riverpod, Bloc, or GetX.
- Navigation: go_router only
- Offline: sqflite for local cache + sync queue
- Barcode: mobile_scanner
- Charts: fl_chart
- Language support: English, Hindi, Telugu via Flutter intl

## Firebase Config
- Project ID: shopsmart-0001
- Initialize with: `flutterfire configure --project=shopsmart-0001`
- Use `firebase_options.dart` auto-generated by flutterfire

## Key Rules
- NEVER create a new Firebase project. Connect to existing shopsmart-0001.
- NEVER use Riverpod, Bloc, or GetX.
- NEVER call Firestore directly from a widget — always through a service.
- ALWAYS show loading indicator while waiting for Firestore.
- ALWAYS queue bills to sqflite when offline and sync when back online.
- ALWAYS use the GST slab system (0%, 5%, 12%, 18%, 28%).
